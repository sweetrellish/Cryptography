# Presentation Reference Pack

This folder collects the presentation planning material created during prep for the AES demo.

Files included:

- `slides_17_24_extract.txt`: raw extracted text from the PDF for slides 17 through 24
- `slides_17_24_cross_reference.md`: mapping from the PDF slides to the simulator features and recommended talking points
- `slides_17_24_rewritten_notes.md`: a cleaner line-by-line speaking version for slides 18 through 24
- `slides_17_24_click_runbook.md`: exact click sequence to use in the app while those slides are on screen
- `presentation_deck_outline.md`: the 6-slide recommended deck structure
- `presentation_scripts.md`: 5-minute and 10-minute scripts
- `presentation_day_checklist.md`: emergency and setup checklist for presentation day

Recommended use:

1. Read `slides_17_24_cross_reference.md` first.
2. Rehearse from `slides_17_24_rewritten_notes.md`.
3. Keep `slides_17_24_click_runbook.md` open during final practice.
4. Use `presentation_day_checklist.md` right before class.
